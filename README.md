# hr-server
没接触过的人事管理系统，参考了江南一点雨的微人事项目  ![微人事](https://github.com/lenve/vhr)

#### 技术栈
  * 数据库：mysql 8.0.16
  * 后端：springboot + mybatis + shiro
  * 前端：vue + iview3.x
